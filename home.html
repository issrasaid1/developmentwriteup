Development: Car shopping React application

The goal of this project was to create a list interface using React. I created an application that allows people to shop for cars. They are able to filter the car that they are interested in by size (small vs SUV) and by type (luxury vs affordable). The user is also able to sort the cars by price from low to high. Finally, they can choose the car(s) they are interested in and add to cart.
Here is the link to the application: Car Shopping. I have also linked my github repo with my code for the app here: Repo.


Screenshot of application

Usability principles

In order to allow optimal usability for the user, I considered a few usability principles when creating this application. For one, I kept visual-hierarchy in mind when designing. For example, I use larger font sizes to indicate headings on the page. I also made the titles for each item in bold so that the user can distinguish between the name of the car model and the rest of its details (e.g. type, size, price, etc). Furthermore, I made my filtering and sorting options acessible through drop down menus, rather than individual buttons. I did this so the user is able to easily see the current state of filtering and sorting. It also allows for a cleaner design as opposed to traditional buttons.

Conclusion

This project allowed me to practice working with frameworks in React.js such as bootstrap. I also learned more about how an effective use of components and states allows for the creation of a clearer architecture. Overall, creating this list interface gave me a good first exposure to desigining web apps using React.
